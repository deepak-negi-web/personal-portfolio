## 🛠 Wanna run on your machine?

1. Clone the project

   ```sh
   git clone https://github.com/deepak-negi-web/personal-portfolio.git
   ```

1. Install the dependencies

   ```sh
   cd personal-portfolio
   yarn
   ```

1. Start the development server

   ```sh
   yarn dev
   ```
